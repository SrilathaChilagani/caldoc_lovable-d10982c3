# CalDoc Product Brief (Launch Track)

## Vision
CalDoc is a fast, reliable telehealth platform for booking, consultation, payments, and follow-up care.

## Launch Goal
Ship a production-ready v1 focused on:
1. Booking + checkout reliability
2. Provider and patient portal stability
3. Fast UX on mobile networks
4. Operational readiness (monitoring, backups, runbooks)

## North Star Metrics
- Booking conversion rate
- Appointment completion rate
- Payment success rate
- P95 page load / API latency
- Weekly active patients/providers

## Non-negotiables
- No data loss
- No broken auth flows
- No silent payment failures
- No uncached heavy assets on hot paths
